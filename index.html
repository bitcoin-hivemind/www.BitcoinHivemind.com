Bitcoin "Smart Contracts" (as Smart as Whatever's Reading Them)

Hivemind is a Peer-to-Peer Oracle Protocol which absorbs accurate data into a blockchain so that Bitcoin-users can speculate in Prediction Markets. Bitcoin Maximal

Planned to be a sidechain of Bitcoin. This means that Bitcoin (and only Bitcoin) is used to create and trade in Markets. A separate coin-type does some pretty fancy things to ensure that a group of self-selected, anonymous, greedy users will always resolve the contract-outcomes honestly.

The Goal: Scientific Revolution Part II

Speaking technically, this project aims to get external data into a blockchain reliably, and use that data efficiently and conservatively.

However, unlike other blockchain-projects, the ultimate purpose is NOT to establish a new payment system (nor to delude the innocent with get-rich-quick nonsense), but instead to...

  • ...overhaul democratic governance (deemphasizing showmanship/PR, highlighting the irrelevance of race/gender/who-candidate-has-slept-with)...
  • ...boost economic growth...
  • ...provide a novel way to efficiently fund public goods, without taxation or coercion...
  • ...cut out all Wall St. / banking middlemen and replace them with tech...
  • ...spearhead the campaign to make Bitcoin the world's internet currency...
  • ...and more!
